# 权限沙箱

确保 Agent 不越权访问文件或执行危险命令。

## 核心模块

- **PathValidator** - 路径校验（realpath 防 symlink 绕过）
- **CommandValidator** - 命令风险识别
- **PermissionManager** - 权限分级管理

## 权限分级

| 操作 | 级别 | 处理方式 |
|------|------|----------|
| 读取文件 | ✅ 自动允许 | 直接执行 |
| 创建文件 | ✅ 自动允许 | 直接执行 |
| 删除文件 | ✅ 自动允许 | 直接执行 |
| 修改文件 | ⚠️ 需确认 | 弹窗确认 |
| 执行命令 | ⚠️ 需确认 | 弹窗确认 |
| 工作区外操作 | 🚫 禁止 | 拒绝执行 |
