## 三、附录

### 3.1 测试工具

- **应用启动**：`npm run electron:dev`
- **构建打包**：`npm run build:win`
- **日志查看**：Chrome DevTools (Ctrl+Shift+I)

### 3.2 常见问题排查

#### 应用无法启动

1. 检查 Node.js 版本（需要 18+）
2. 检查依赖是否安装（`npm install`）
3. 检查端口是否被占用

#### Agent 无法初始化

1. 检查 Claude CLI 是否安装（`claude --version`）
2. 检查 Claude Profile 配置是否正确
3. 检查网络连接

#### 权限弹窗不显示

1. 检查权限 IPC 是否注册
2. 检查 preload 白名单是否包含权限通道
3. 检查 PermissionManager 是否初始化

### 3.3 测试环境清理

1. 删除测试项目中的临时文件
2. 清理 electron-store 配置
3. 重启应用
